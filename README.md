# Health Terminal (BMI Calculator)

[RU] Простой и эффективный консольный калькулятор индекса массы тела (ИМТ).
[EN] A simple and effective CLI Body Mass Index (BMI) calculator.

--------------------------------------------------------------------------

## Features / Особенности

-  **Bilingual / Двуязычный**: Full support for English and Russian.
-  **Visuals / Визуал**: Color-coded terminal output for better UX.
-  **Robust / Надежность**: Error handling for invalid inputs (strings/symbols).
-  **Smart Units / Умные замеры**: Auto-conversion for height (supports both 1.80m and 180cm).
